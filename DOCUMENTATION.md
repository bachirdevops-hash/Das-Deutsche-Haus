## 📋 SECTION 1 — ملخّص المشروع

### الهدف
**Das Deutsche Haus** هو موقع تعليمي وثقافي ثنائي اللغة (عربي/ألماني) يعمل كجسر بين سوريا 🇸🇾 وألمانيا 🇩🇪. يقدّم:
- 📚 كورسات اللغة الألمانية المعتمدة (A1 → C2 وفق CEFR)
- 🏆 امتحانات telc الرسمية
- 🛠️ التدريب المهني (Ausbildung) في ألمانيا
- ✈️ استشارات تأشيرات الدراسة والعمل
- 🇩🇪 خدمات سياحية للزوار الألمان (German Visitors)
- 📰 مدوّنة + 🗓️ نشاطات

### الرابط الحالي
```
https://telc-academy.preview.emergentagent.com
```

### التقنيات المستخدمة
| الطبقة | التقنية |
|---|---|
| Frontend | Next.js 14 (App Router) + React 18 |
| Styling | Tailwind CSS + shadcn/ui |
| Font | IBM Plex Sans Arabic + Inter |
| Backend API | Next.js Route Handlers (catch-all) |
| Database | MongoDB |
| File Storage | Cloudinary (Signed Uploads) |
| Email | SendGrid (مُعدّ، يحتاج مفتاح) |
| Auth | Custom JWT (HTTP-Only Cookies) + scrypt password hashing |

### الإحصائيات
- 📄 **9 صفحات عامة** (Home, Courses, telc, Vocational, Travel, About, Contact, Blog, Activities, German Visitors)
- 👥 **4 أدوار** (Super Admin, Manager, Teacher, Student)
- 🔌 **80+ API endpoint**
- 🗂️ **15+ MongoDB collection**
- 🌐 **2 لغة كاملة** (AR + DE)

## 👑 SECTION 2 — دليل المدير (Super Admin)

### أ) تسجيل الدخول
1. افتح الموقع وانقر على **"دخول"** أعلى يسار/يمين الشاشة.
2. أدخل البريد الإلكتروني وكلمة المرور.
3. ستُحوَّل تلقائياً إلى لوحة الإدارة.

> **💡 ملاحظة:** بيانات الدخول الافتراضية في القسم 6.

### ب) إضافة كورس جديد
1. من الهيدر → **"لوحة الإدارة"** (الزر الذهبي).
2. التبويب الافتراضي **"الإحصائيات"** يعرض النظرة العامة.
3. الكورسات تُدار حالياً عبر seed تلقائي (A1-C2). لإضافة/تعديل، استخدم تبويب **"المستخدمون"** → تعديل كورسات الطلاب.

### ج) إضافة امتحان telc
1. الكورسات وامتحانات telc تُملأ من الـseed تلقائياً.
2. للتعديل اليدوي، ستحتاج إلى الوصول لقاعدة البيانات أو إضافة CRUD لاحقاً.

### د) إدارة المستخدمين وتغيير الأدوار
1. لوحة الإدارة → تبويب **"المستخدمون"**.
2. اضغط أيقونة القلم بجانب أي مستخدم.
3. يمكنك تغيير: الاسم، الدور (super_admin/manager/teacher/student)، تعطيل الحساب، تعيين كلمة مرور جديدة.
4. **ملاحظة أمنية:** لا يمكنك حذف حسابك الخاص.

### هـ) تعيين معلم لكورس
1. لوحة الإدارة → تبويب **"تعيين معلمين"**.
2. اختر مستخدماً بدور `teacher`.
3. حدّد الكورسات المراد تعيينها.
4. اضغط حفظ.

### و) إدارة صفحة الزوار الألمان (`/german-visitors`)
1. لوحة الإدارة → تبويب **"🇩🇪 الزوار الألمان"** (خلفية ذهبية).
2. تتوفّر **11 تبويب فرعي:**
   - 📦 الباقات السياحية
   - ✨ التجارب
   - 💬 شهادات الزوار
   - ❓ الأسئلة الشائعة
   - 🎴 البطاقات التعليمية (Flashcards)
   - 🖼️ المعرض
   - 📅 الحجوزات
   - 🛎️ طلبات الخدمة
   - 🚨 الأرقام الطارئة
   - 🔥 لماذا سوريا؟ (Why Cards)
   - ⚙️ إعدادات الصفحة (Hero, CTA, WhatsApp, SEO، تفعيل/تعطيل الأقسام)

كل تبويب يحتوي على CRUD كامل (إضافة/تعديل/حذف).

### ز) إدارة المدوّنة
1. لوحة الإدارة → تبويب **"📰 المدوّنة"** (خلفية حمراء).
2. اضغط **"+ مقال جديد"** أعلى الصفحة.
3. املأ:
   - العنوان (مطلوب)
   - Slug (تلقائي إن تُرك فارغاً)
   - الفئة (تعليم، أخبار، نصائح، الحياة في ألمانيا، نجاحات، أخرى)
   - **لغة المقال (🇸🇾 ar / 🇩🇪 de)** ← مُهم!
   - صورة الغلاف (Cloudinary)
   - الكاتب وصورته
   - تاريخ النشر (يدعم الجدولة المستقبلية)
   - المحتوى (Rich Text Editor)
   - المقتطف (يُولَّد تلقائياً)
   - وصف SEO
   - الحالة (مسودّة / منشور / مخفي)
4. اضغط **حفظ**.
5. للتصفية: استخدم فلاتر الفئة، الحالة، اللغة، أو البحث.

### ح) إدارة النشاطات
1. لوحة الإدارة → تبويب **"🗓️ النشاطات"** (خلفية سوداء).
2. اضغط **"+ نشاط جديد"**.
3. املأ التفاصيل:
   - العنوان والنوع (ورشة/محاضرة/احتفال/رحلة/يوم مفتوح/دورة مكثّفة)
   - صورة الغلاف، التاريخ والوقت، المكان، رابط Maps
   - السعر (مجاني/مدفوع + العملة)
   - **يحتاج تسجيل مسبق** (مفتاح تبديل):
     - إن نعم → يظهر حقل **إجمالي المقاعد** + موعد إغلاق التسجيل
   - الحالة (مسودّة/منشور/مخفي/ملغى)
4. للوصول إلى التسجيلات: اضغط أيقونة 👥 الزرقاء بجانب أي نشاط.
   - عرض جدول التسجيلات
   - تعديل الحالة (قيد الانتظار/مؤكَّد/ملغى)
   - واتساب مباشر بنقرة واحدة (أيقونة المحادثة الخضراء)
   - حذف التسجيل (يُسترجع المقعد تلقائياً)
   - تصدير CSV لجميع التسجيلات
   - إضافة تسجيل يدوي

### ط) مراجعة الحجوزات والخدمات (الزوار الألمان)
1. لوحة الإدارة → "🇩🇪 الزوار الألمان" → **"الحجوزات"** أو **"طلبات الخدمة"**.
2. كل صف يحوي معلومات الزائر + حالة (جديد/تواصل تم/مؤكَّد/مكتمل).
3. تعديل سريع للحالة + إضافة ملاحظات إدارية.

### ي) تعديل أرقام الطوارئ والسفارات
1. لوحة الإدارة → "🇩🇪 الزوار الألمان" → **"الأرقام الطارئة"**.
2. CRUD كامل: الاسم، النوع (سفارة/شرطة/إسعاف)، الرقم، البريد، التفعيل.

## 👨‍🏫 SECTION 3 — دليل المعلم

### تسجيل الدخول
- استخدم البريد وكلمة المرور التي زوّدك بها المدير.
- ستُحوَّل تلقائياً إلى **"لوحة المعلم"**.

### إضافة جلسة Zoom
1. اختر الكورس من بطاقات لوحة المعلم.
2. تبويب **"الجلسات"**.
3. أضف عنوان الجلسة، رابط Zoom، التاريخ والوقت.

### رفع ملف للطلاب
1. تبويب **"المواد"** داخل الكورس.
2. ارفع PDF/فيديو/صور (Cloudinary).
3. الطلاب المسجّلون في الكورس يرونه فوراً.

### إضافة درجات
1. تبويب **"الدرجات"**.
2. اختر الطالب، نوع الاختبار، الدرجة، التعليق.

### تسجيل الحضور والغياب
1. تبويب **"الحضور"**.
2. حدّد التاريخ، حالة كل طالب (حاضر/غائب).

### مراسلة الطلاب
- **رسالة عامة:** تبويب الرسائل → إعلان للجميع.
- **محادثة خاصة:** اضغط على اسم الطالب → فتح محادثة.

## 🎓 SECTION 4 — دليل الطالب

### إنشاء حساب
1. الهيدر → **"إنشاء حساب"** (الزر الأحمر).
2. أدخل: الاسم، البريد، الجوال، كلمة المرور.
3. سجّل الدخول وستُحوَّل إلى **"لوحتي"**.

### التسجيل في كورس
1. اذهب إلى **"التعليم → الكورسات"** من الهيدر.
2. اختر المستوى (A1-C2).
3. اضغط **"سجّل الآن"**.
4. أكّد الحجز.

### الدخول إلى غرفة الكورس
1. **"لوحتي"** → تبويب **"كورساتي"**.
2. اضغط على الكورس → فتح غرفة الكورس.
3. تبويبات: **المواد / الجلسات / درجاتي / محادثة المعلم**.

### عرض الدرجات
- **"لوحتي"** → اختر الكورس → تبويب **"درجاتي"**.

### تحميل المواد
- داخل غرفة الكورس → تبويب **"المواد"** → اضغط أيقونة التنزيل.

### التواصل مع المعلم
- داخل غرفة الكورس → تبويب **"محادثة المعلم"** → اكتب رسالتك.
- ستظهر إشعارات عند الرد عبر **🔔** في الهيدر.

### حجز امتحان telc
1. الهيدر → الزر الأحمر **"امتحانات telc"**.
2. اختر الامتحان (A1/A2/B1/B2/C1).
3. اضغط **"احجز"** + ادفع الرسوم.

## 🚀 SECTION 5 — قائمة الميزات الكاملة

### الواجهة العامة
| الميزة | الوصف |
|---|---|
| 🌍 ثنائية اللغة | عربي/ألماني مع RTL/LTR تلقائي |
| 🎨 Navbar متقدّم | 4 dropdowns + زر telc أحمر بارز + DE button بإطار ذهبي |
| 🔔 إشعارات | جرس Realtime مع عداد |
| 📱 WhatsApp Float | زر تواصل مباشر |
| 🎯 SEO | Meta tags لكل صفحة |
| 📰 مدوّنة كاملة | فلاتر، بحث، صفحات، AR/DE، شارة لغة، Rich Text Editor |
| 🗓️ نشاطات | فلاتر، تسجيل عام مع Anti-Oversell، CSV export |
| 🇩🇪 صفحة الزوار الألمان | 12+ قسم: باقات، تجارب، FAQ، Flashcards، Gallery Lightbox، نماذج، طوارئ |

### نظام الصلاحيات (RBAC)
- **Super Admin:** صلاحيات كاملة (CRUD المستخدمين + كل المحتوى)
- **Manager:** نفس Super Admin ما عدا إدارة المستخدمين
- **Teacher:** Course Rooms المسندة فقط
- **Student:** عرض كورساته + التواصل + الدرجات

### ميزات أمنية
- 🔒 JWT in HTTP-Only Cookies
- 🛡️ Rate Limiting (5/دقيقة لـ login + التسجيلات)
- 🔐 Password hashing (scrypt + salt)
- ⚛️ Atomic operations لـ Anti-Oversell
- 📜 Activity Logs لكل عملية إدارية
- 🚫 Email enumeration prevention في "نسيت كلمة المرور"

### ميزات Cloudinary
- ⚡ رفع موقّع (signed) لمنع إساءة الاستخدام
- 📁 Folder whitelist حسب الدور
- 🗑️ حذف من الموقع يحذف من Cloudinary

### ميزات إدارية
- 📊 Dashboard إحصائي (إيرادات، تسجيلات، حسب الدور)
- 📥 تصدير CSV (تسجيلات النشاطات)
- 🔍 بحث وفلترة على كل الجداول
- 📜 سجل النشاط (آخر 500 عملية)
